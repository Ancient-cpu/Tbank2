# T-Библиотека (T-Library)

Консольное приложение для управления личной библиотекой книг. Позволяет добавлять книги, управлять их статусами (прочитана/не прочитана), вести список избранного, а также искать и фильтровать книги по различным параметрам. 

Проект разработан на **Python** с использованием встроенных библиотек. Все данные надежно сохраняются в локальный JSON-файл и автоматически загружаются при следующем запуске.

## 🚀 Основной функционал

*   **Добавление книг**: Запись названия, автора, жанра, года и описания. Каждой книге автоматически присваивается уникальный ID.
*   **Умный просмотр**: 
    *   Сортировка по названию, автору или году издания.
    *   Фильтрация по жанру или статусу прочтения.
*   **Поиск**: Поиск по ключевым словам в названии, имени автора или описании книги.
*   **Избранное**: Возможность добавлять книги в список любимых и просматривать этот список отдельно.
*   **Управление статусами**: Смена статуса книги "прочитана / не прочитана" в один клик (ввод ID).
*   **Удаление**: Безопасное удаление книг из базы.
*   **Сохранение**: Данные хранятся в файле `library_data.json`.
